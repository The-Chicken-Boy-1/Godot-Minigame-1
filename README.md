# Godot Minigame 1
## Devlog
I think the relationship between Nodes and Scenes in Godot is kind of like blocks and builds in Minecraft. Each individual block in Minecraft is like a node. On its own, a single dirt block or stone block doesn’t really do much. It just exists and has a basic function. But when you start placing blocks together in a certain way, you can create something bigger, like a house, a castle, or even a redstone machine. That full build is like a scene in Godot. It’s made up of smaller parts that work together to create something useful.

To explain it more, nodes in Godot each have their own specific job, like playing a sound, or detecting collisions. By themselves, they’re just small pieces. But when you combine multiple nodes into one organized structure, you create a scene, like a player character or an enemy. In the same way, in Minecraft, you combine different types of blocks to create a full structure that actually serves a purpose. So basically, nodes are the individual blocks, and scenes are the finished builds that bring everything together into something that works in your game.
